# Documentacao

Esta pasta concentra os materiais de apoio do projeto.

## Estrutura

- `architecture/`: visao estrutural, planos de arquitetura e organizacao de modulos
- `flows/`: fluxos funcionais e visoes de processo
- `guides/`: guias operacionais, conectividade e implementacoes complementares
- `plans/`: planos de teste, ideias de evolucao e materiais de planejamento
- `reference/`: anotacoes, referencias internas e arquivos auxiliares
- `tcc/`: materiais relacionados ao TCC

Os arquivos `er.mmd`, `er.png` e `OO-Relational-Mapping.md` continuam na raiz de `docs/` por serem referencias tecnicas gerais do modelo do sistema.
