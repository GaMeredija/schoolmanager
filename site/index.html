School Manager

Sistema escolar web pensado para centralizar a rotina academica em um unico ambiente, com perfis de administrador, diretor, coordenador, professor e aluno.

Ver repositorio Como rodar

Visao geral

Uma base unica para a gestao escolar

O projeto organiza turmas, disciplinas, notas, frequencia, calendario, atividades, materiais, relatorios e comunicacao interna em uma experiencia web unica.

Essencia da publicacao

  • Frontend React + Vite
  • Backend Node.js + Express
  • Banco SQLite
  • Perfis e fluxos academicos integrados

Modulos

O que esta organizado dentro do sistema

Usuarios e perfis

Controle de acesso para administracao, diretoria, coordenacao, docentes e estudantes.

Academico

Turmas, disciplinas, calendario, periodos letivos, matriculas e acompanhamento da rotina escolar.

Avaliacao

Notas, exames, frequencia, atividades, entregas e acompanhamento do desempenho.

Comunicacao

Mensagens, notificacoes, materiais e fluxos de acompanhamento entre equipe escolar e alunos.

Arquitetura

Como o projeto esta estruturado

01

Client

Interface web feita em React com build via Vite.

02

Server

API e regras de negocio em Node.js e Express.

03

Shared

Schema compartilhado entre frontend, backend e banco.

Execucao

Como rodar o sistema completo localmente

Esta pagina no GitHub Pages funciona como vitrine. Para usar o sistema completo com API e banco, rode o projeto localmente. 

npm ci
npm run dev

Acesso local padrao: http://localhost:3001
