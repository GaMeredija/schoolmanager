# Scripts Auxiliares

Esta pasta organiza os utilitarios que antes estavam soltos na raiz do repositorio.

## Estrutura

- `admin/`: scripts de apoio a administracao de usuarios
- `database/`: manutencao e diagnostico de banco
- `testing/`: dados de teste e simulacoes de fluxo

Esses scripts nao fazem parte da inicializacao normal da aplicacao web. Eles servem como apoio operacional e de manutencao.
